# Tagbox

Authentic garments for the digital underground. A fashion-brand landing page built with TanStack Start, React 19, Tailwind CSS v4, and Vite 7 — designed to feel editorial, sensual, and future-forward.

![Tagbox hero](src/assets/hero.jpg)

## Stack

- **Framework:** TanStack Start v1 (file-based routing under `src/routes/`)
- **UI:** React 19 + Tailwind CSS v4 (CSS-first config in `src/styles.css`)
- **Build:** Vite 7
- **Runtime:** Vercel (Nitro `vercel` preset — also portable to Node, Cloudflare, Netlify by switching the preset)
- **Package manager:** Bun

## Design system

Defined in `src/styles.css` with semantic tokens — never hardcoded in components.

| Token         | Value     | Use                         |
| ------------- | --------- | --------------------------- |
| `--color-onyx`      | `#0a0a0a` | Primary background          |
| `--color-onyx-soft` | `#141414` | Secondary surfaces          |
| `--color-acid`      | `#dfff00` | Accent / CTAs / highlights  |
| `--color-platinum`  | `#e5e5e5` | Body text                   |
| `--font-display`    | Syne 700/800 | Display headlines        |
| `--font-sans`       | Inter 300–600 | Body                    |

Custom utilities: `grain`, `animate-reveal-up`, `animate-reveal-clip`, `animate-marquee`, `animate-pulse-dot`.

## Getting started

```bash
bun install
bun run dev
```

Open <http://localhost:8080>.

### Other scripts

```bash
bun run build       # production build
bun run start       # serve production build
bun run lint        # eslint
```

## Project structure

```
src/
├── assets/              # Hero + product imagery (imported as ES modules)
├── routes/
│   ├── __root.tsx       # Root shell, head, fonts, providers
│   └── index.tsx        # Tagbox homepage
├── styles.css           # Tailwind v4 entry + design tokens + utilities
├── router.tsx
└── start.ts
public/
└── robots.txt
```

`src/routeTree.gen.ts` is auto-generated by the TanStack Router Vite plugin — do not edit.

## Sections on the homepage

1. **Fixed nav** with `mix-blend-difference` so it adapts over any background
2. **Hero** — full-bleed editorial image, kinetic clip-path reveal of the wordmark, live status annotation
3. **Marquee strip** — infinite scroll of brand keywords
4. **Current Drop** grid — three product cards with hover scale + acid-yellow CTAs
5. **Manifesto** — full-bleed acid-yellow editorial block with rotating culture image
6. **Newsletter** — email capture for next drop notifications
7. **Footer** — oversized brand mark + link columns



### Switching targets

Edit `vite.config.ts` and change the Nitro preset:

```ts
nitro: { preset: "vercel" }       // default
nitro: { preset: "node-server" }  // self-host (node .output/server/index.mjs)
nitro: { preset: "cloudflare-module" }
nitro: { preset: "netlify" }
```
